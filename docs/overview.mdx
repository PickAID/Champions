# Champions-Unofficial

This mod is an unofficial port of the Champions mod for Minecraft Forge 1.20.1. This version is currently under limited testing and may contain bugs.

If you encounter any issues, please report them to us instead of TheIllusiveC4.

<Callout variant="warning">
    This port is permitted by TheIllusiveC4 and development will stop once an official update is released. However, enhancement updates, such as additional ranks, are planned for this version from 1.18.2. When the official version of Champions are released, this mod will be rewritend as a addons for Champions.
</Callout>

Champions is a mod that adds elite mobs of different rarities, with enhanced stats, extra abilities, and extra loot. These champions can spawn from any hostile mob in the game.

<Callout>
    Champions now support kubejs, that allow you to register your custom affix work with champions.
</Callout>
